HeliosSuite

➕ Zainstaluj w Tampermonkey

Jeśli przeglądarka pobiera plik zamiast instalować – otwórz link RAW i zaakceptuj instalację w Tampermonkey.

Co dostajesz?

  • Zakładka HeliosSuite w ustawieniach Grepolis
  • Motywy: Classic / Remaster / Piracki / Dark
  • Naprawione okna/z-index (koniec „zjeżdżania”)
  • HeliosPulse: presence + dzienny raport (BBCode)
  • Czat (alpha) – wersja lokalna

Szybki start

  1. Zainstaluj skrypt (przycisk powyżej)
  2. Odśwież Grepolis
  3. Kliknij ikonę ☀️ po lewej lub wejdź w Ustawienia ⇒ HeliosSuite
